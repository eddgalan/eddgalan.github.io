PriceShop

Este es el primer proyecto 'grande' que comencé a desarrollar solo. Este es un sistema para poder llevar el control de ventas por catálogo. Aún no está terminado pero ya maneja varias funciones interesantes. El sitio web está montado en un host gratuito, está programado con PHP, jQuery, hace peticiones AJAX, tiene varios CRUD y maneja sesiones. También usa DataTables, tiene una Base de Datos My SQL, el sitio no es responsivo al 100% pero no se tiene una mala experiencia en móviles, usa la arquitectura MVC y tiene algunas APIs que son consumidas con AJAX. Para entrar al sitio es necesario ingresar usuario y contraseña. Si está interesado en ver el sitio puede solicitarme el acceso, ya que por seguridad del sitio, no dejaré los accesos aquí.
