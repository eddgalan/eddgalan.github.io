Descripción

Este proyecto consiste en un sistema de facturación para CFDIs Versión 3.3. El sistema permite la generación de facturas y el timbrado de estas mismas usando un PAC. Está desarrollado con PHP puro (sin ningún Framework), utiliza la arquitectura MVC. Como gestor de base de datos utiliza MySQL/MariaDB. Las tecnologías que usé para desarrollarlo son las siguientes: javascript, jQuery, Bootstrap, composer para instalar MPDF y MailGun, TCPDF, ChartJS para la creación de algunos gráficos sencillos.
